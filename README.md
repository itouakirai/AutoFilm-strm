# AutoFilm-strm
**加速strm文件生成**
## 注意
只魔改了autofilm.py文件，只测试了单生成strm，其余未测，
如果想更快或更慢，请修改文中两处（max_workers=5）中的数值，默认为5，参考自己配置修改

详情见[Akimio的博客](https://blog.akimio.top/posts/1031/#使用教程)
## 已知问题
